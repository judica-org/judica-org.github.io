software.

judica is creating software that will have a remarkable impact on the bitcoin ecosystem. we endeavor to provide these to the community openly. these applications are available without any new features added to bitcoin, and can leverage new features as they become available.

this page is an index of some of the projects currently in development at judica. consider joining us if you want to work on them!

judica dao

a one-size-fits-all implementation of a dao on bitcoin with support for the features required by a multitude of use cases.

sapio

a flexible and powerful smart contracting language that enables stateful smart contracts.

a runtime engine for federated bitcoin smart contracts.
